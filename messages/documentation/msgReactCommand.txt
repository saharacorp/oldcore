Message Reaction Command
This command is used as a show-and-tell of what listeners do. It will automatically react to messages sent by a particular person
This is an owner-only command

Arguments:
 * subcommand - Action for the reactions
   * help - Displays the help page for this command
   * status - Displays status of message reactions
   * toggle - Toggles message reactions
   * setemote - Sets the emote to react with
   * settarget - Sets the target to listen for
 * value - Area for subcommands which require extra data
   * setemote requires this to be an EmojiResolvable
   * settarget requires this to be a mention. (To mention someone with only their ID, just surround the id with <@id goes here in between these braces>)
